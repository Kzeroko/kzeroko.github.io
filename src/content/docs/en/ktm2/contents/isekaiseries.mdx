# 📦Isekai Core

Isekai Core is the core module of the Isekai Series. You can think of it as the prerequisite for all other Isekai Series modules, as they all depend on it. Isekai Core provides some common API code for the Isekai Series.

---

# ⚔️Isekai Expansion

Isekai Expansion is the main expansion module for the KTM2 modpack. Its predecessor, Isekai Weaponry, has been completely rewritten and specialized for the KTM2 modpack. Isekai Expansion bridges some of the important mods in the pack to create new content, generally focusing on player skills, attributes, and combat. It also includes a large number of material items for KTM2, which usually have no function other than for in-game trading, loot, or crafting recipes.

---

# 🛠️Isekai Tweaks

Isekai Tweaks is the most miscellaneous module in the KTM2 Isekai Series. It involves using mixins to modify and improve the content of various mods. Some modifications that are not suitable for other expansions are also included in Isekai Tweaks. If a new feature involves multiple mods, then Isekai Tweaks is the right place for it.

---

# 🏘️Isekai Structures

Isekai Structures is the building structure expansion module in the KTM2 modpack. Expansion features related to structures are also categorized into this module.

---

# 🍽️Isekai Cuisine

Isekai Cuisine is the food and drink item expansion module in the KTM2 modpack. It also includes a fishing module and compatibility for most of the Farmer's Delight food recipes.

---

# 🎼Isekai Melody

Isekai Melody includes an override for the original Minecraft soundtrack and a feature for unique background music in structures to create a more immersive world interaction. The music used is licensed under the CC BY 4.0 license (see the CREDITS file in the module for details).
